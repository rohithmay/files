def kpss_test(series, **kw):    
    statistic, p_value, n_lags, critical_values = kpss(series, **kw)
    # Format Output
    print(f'KPSS Statistic: {statistic}')
    print(f'p-value: {p_value}')
    print(f'num lags: {n_lags}')
    print('Critial Values:')
    for key, value in critical_values.items():
        print(f'   {key} : {value}')
    print(f'Result: The series is {"not " if p_value < 0.05 else ""}stationary')

from statsmodels.tsa.stattools import adfuller
def adf_test(timeseries):
    #Perform Dickey-Fuller test:
    print ('Results of Dickey-Fuller Test: \n')
    dftest = adfuller(timeseries, autolag='AIC')
    dfoutput = pd.Series(dftest[0:4], index=['Test Statistic','p-value','#Lags Used','Number of Observations Used'])
    for key,value in dftest[4].items():
       dfoutput['Critical Value (%s)'%key] = value
    print (dfoutput)

The ADF test posits:
$H_0$ : The time series is not stationary.
$H_1$ : The time series is stationary.

Our objective is to reject the null hypothesis when the p-value is less that 0.05 and ensure stationarity of the time series data.

The ADF test shows a p value <0.05. Reject Null. Returns time series data is stationary.

The model fitted above is ARMA(2,0) = AR(2).
Note that the first lag term in AR(2) turns out to be insignificant.
There for the AR model corresponding to this is:

$$ \hat{r}_t = 0.0187 -0.0868 r_{t-2}$$¶

$$ r_t = 0.0187 -0.0868 r_{t-2} + \epsilon_{t}$$¶

Ljung-Box Test¶

$H_0$ : The residuals are uncorrelated.
$H_1$ : The residuals are correlated.

Since p value > 0.05, accept the null that the returns time series values are uncorrelated. There is however an indication of volatility clustering. Hence we adopt a GARCH model.

The fitted GARCH(1,1) model is:

$$ \epsilon_t = \sigma_t \omega_t $$¶

$$ \sigma_t^2 = \alpha_0 + \alpha_1 \epsilon_{t-1}^{2} + \beta_1 \sigma_{t-1}^{2} $$¶

$$ r_t = \mu + \epsilon_t $$¶

Checking the validity of the GARCH model¶

The following test are conducted:

  1. The standardized residuals from the GARCH model should approach normal distribution. Use Shapiro-Wilk test and Jarque-Bera normality test. Histogram of the residuals is also a good visual tool to check normality.
  2. The standardized squared residuals from the GARCH model should not be autocorrelated. Use Ljung-box Q-statistic for this purpose.
  3. ARCH LM test on the residuals can also be conducted to check for remaining ARCH effects in the residuals.
